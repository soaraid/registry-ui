Place brand assets for the UI in this directory.

Recommended filenames:

- `logo.svg`
- `logo-light.svg`
- `logo-dark.svg`
- `logo-mark.svg`

Usage notes:

- Use `logo.svg` if you only have one logo file.
- Use `logo-light.svg` for dark backgrounds.
- Use `logo-dark.svg` for light backgrounds.
- Use `logo-mark.svg` for a compact square mark if needed later.

These files will be available from:

- `/brand/logo.svg`
- `/brand/logo-light.svg`
- `/brand/logo-dark.svg`
- `/brand/logo-mark.svg`
